# GeburtstagFinal

Eine einfache, offline Geburtstags-Erinnerungs-App für Android.

## Features
- Geburtstage hinzufügen, bearbeiten, löschen
- Suche
- Benachrichtigungen (inkl. nach Geräte-Neustart)
- Import/Export als JSON

## Datenschutz
Diese App arbeitet komplett offline.
Es werden keine Daten gesammelt oder übertragen.
